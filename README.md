# Python网易云热门歌单可视化大屏

## 核心功能
- 歌单类型分析
- 分享量与评论数分析
- 歌单歌曲数量范围
- 年份歌单轮播
- 最受欢迎歌单
- 歌单收藏量变化
- 语种类型歌单播放量

## 适用场景
- 网易云音乐数据分析
- 音乐市场趋势研究
- 音乐推荐系统开发
- 教育与培训使用

## 技术特点
- 使用 Flask 框架实现后端
- 数据存储于 MySQL 数据库
- 前端采用 HTML、CSS 和 JavaScript 技术
- 可视化效果丰富，交互性良好
